# License4J
启动项目前请在pom文件配置好数据库连接配置信息

#数据库的配置信息
```spring:
    datasource:
        url: jdbc:mysql://
        username: 
        password: 
        driver-class-name: com.mysql.cj.jdbc.Driver
```
